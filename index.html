Project 1: Primary Color Blinking

Date: 2023-09-20

This assignment involves making a red, green, and blue LED blink one by one in repetition. This requires wiring LEDs independently on a single breadboard, then writing a loop with wait cycles to turn them on and off. While simple, this lab helps demonstrate correct, independent wiring and custom script writing.

Hardware Components

  1. ESP32 WROVER (x1)
  2. GPIO Extension Board (x1)
  3. Breadboard (x1)
  4. Distinctly Colored LEDs (x3)
  5. Resistor, 220 Ω (x3)
  6. Jumper M/M (x6)

Project 2: NOR Latch

Date: 2023-09-20

This assignment involves using buttons to create a NOR-latch. This circuit usually involves NOR gates, but here it is implemented in software. Once an input is given, the output "latches" and will not change unless an off signal is given. Then an input from the on signal is required to turn it on again. In this circuit, the input signals are given from buttons and the output is an LED.

Hardware Components

  1. ESP32 WROVER (x1)
  2. GPIO Extension Board (x1)
  3. Breadboard (x1)
  4. LED (x1)
  5. Resistor, 220 Ω (x1)
  6. Resistor, 10 kΩ (x4)
  7. Jumper M/M (x6)

Project 3: Light Bar Patterns

Date: 2023-09-27

This assignment involves displaying patterns on a segmented LED bar graph in response to buttons. Two buttons each produce a unique behaviour: One produces a light that moves in a "modulus" fashion. The other produces a similar movement with an added blur effect. This project combines knowledge of buttons and the LED bar from the tutorials. It also reviews how to use Pulse Width Modulation (PWM) as an alternative to digital-analog conversion.

Project 4: Rainbow Button

Date: 2023-09-27

This assignment involves displaying different colors of light using two different RGB formats. When a button is pressed, an RGBLED and a NeoPixel array must display the same color. This color changes every time the button is pressed. This project reviews how to connect these two RGB components correctly. Furthermore, the Python code for these two components is completely different, so this serves as a good exercise to compare the two.

Project 5: Song Generator

Date: 2023-09-27

This assignment involves playing songs using a passive buzzer. Buzzers need transistors to amplify the current provided to them, which is easy to wire incorrectly. Another challenge in this lab was to translate notes into frequencies for the buzzer to use. This becomes fairly straightforward if one uses mathematics to calculate the frequencies. Each note is separated by a twelfth root of two, and A4 is 440 Hz.

Hardware Components

  1. ESP32 WROVER (x1)
  2. GPIO Extension Board (x1)
  3. Breadboard (x1)
  4. RGBLED (x1)
  5. Push Button (x2)
  6. Passive Buzzer (x1)
  7. NPN Transistor (x1)
  8. Resistor, 220 Ω (x1)
  9. Resistor, 1 kΩ (x1)

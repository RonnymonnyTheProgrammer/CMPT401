Project 1: Primary Color Blinking

Date: 2023-09-20

This assignment involves making a red, green, and blue LED blink one by one in repetition. This requires wiring LEDs independently on a single breadboard, then writing a loop with wait cycles to turn them on and off. While simple, this lab helps demonstrate correct, independent wiring and custom script writing.

Hardware Components

  1. ESP32 WROVER (x1)
  2. GPIO Extension Board (x1)
  3. Breadboard (x1)
  4. Distinctly Colored LEDs (x3)
  5. Resistor, 220 Ω (x3)
  6. Jumper M/M (x6)

Project 2: NOR Latch

Date: 2023-09-20

This assignment involves using buttons to create a NOR-latch. This circuit usually involves NOR gates, but here it is implemented in software. Once an input is given, the output "latches" and will not change unless an off signal is given. Then an input from the on signal is required to turn it on again. In this circuit, the input signals are given from buttons and the output is an LED.

Hardware Components

  1. ESP32 WROVER (x1)
  2. GPIO Extension Board (x1)
  3. Breadboard (x1)
  4. LED (x1)
  5. Resistor, 220 Ω (x1)
  6. Resistor, 10 kΩ (x4)
  7. Jumper M/M (x6)

Project 3: Light Bar Patterns

Date: 2023-09-20

This assignment involves displaying patterns on a segmented LED bar graph in response to buttons. Two buttons each produce a unique behaviour: One produces a light that moves in a "modulus" fashion. The other produces a similar movement with an added blur effect. This project combines knowledge of buttons and the LED bar from the tutorials. It also uses how to use Pulse Width Modulation (PWM) in custom code.

Hardware Components

  1. ESP32 WROVER (x1)
  2. GPIO Extension Board (x1)
  3. Breadboard (x1)
  4. LED Bar Graph (x1)
  5. Resistor, 220 Ω (x10)
  6. Resistor, 10 kΩ (x4)
  7. Jumper M/M (x14)
